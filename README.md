Portfólio — Vitor Bugallo

Este projeto é um portfólio pessoal desenvolvido com **React + Vite** e estilizado com **TailwindCSS**.

Funcionalidades
- Exibição de projetos com barra de busca dinâmica.
- Formulário de contato funcional (exibe os dados via `alert`).
- Rodapé com ícones interativos (GitHub e LinkedIn).
- Design moderno, responsivo e de fácil navegação.

Tecnologias Utilizadas
- **React 19**
- **Vite**
- **Tailwind CSS**
- **React Icons**
